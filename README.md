🧠 AI for Digital Fraud Detection & Geospatial Crime Intelligence
📌 Overview

This project is an AI-powered system designed to detect and analyze digital fraud patterns along with geospatial crime intelligence. It provides map-based visualization of crime hotspots, identifies high-risk (red zone) areas, and supports data-driven decision-making for safety and investigation purposes.

The system combines machine learning concepts + geospatial analytics + visualization dashboard to improve crime monitoring efficiency.

🚀 Features
🔍 AI-based fraud pattern detection (rule-based / ML-ready module)
📍 Geospatial crime mapping using interactive maps
🔴 Crime hotspot visualization (heatmap / clustering)
⚠️ Red zone identification for high-risk areas
📊 Data-driven analytics dashboard
🎨 UI with pink-themed interface (custom UI design)
🗺️ Location-based decision support system
🛠️ Tech Stack
Programming Language: Python / JavaScript (based on implementation)
Frontend: HTML, CSS, JavaScript
Backend (optional): Flask / Django
Data Processing: Pandas, NumPy
Visualization: Folium / Matplotlib / Plotly
Maps API: Leaflet / Google Maps API
